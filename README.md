## 🎯 项目概述

**VideoForge** 是新一代**自主内容生产系统**，通过**定时生产**、**智能素材管理**和**多平台分发**，彻底革新短视频创作流程。

与传统视频编辑工具不同，VideoForge 作为**全自动内容工厂**运行 —— 7×24小时不间断地生产、优化和分发视频内容，无需人工干预。

### 核心优势

| 能力 | 传统工具 | VideoForge |
|------|----------|------------|
| **生产模式** | 手动，逐个制作 | 自主，批量生产 |
| **定时调度** | 无 | 基于Cron，定时间隔 |
| **素材管理** | 手动选择 | AI驱动，分层存储 |
| **平台分发** | 手动上传 | 自动化，多平台 |
| **扩展性** | 有限 | 无限 |
| **单视频成本** | ¥350-1400 | ¥0.07-0.35 |

### ⏱️ 定时生产系统

**问题**：内容创作者80%的时间花在重复性生产任务上。

**解决方案**：自主调度引擎，让内容生产进入"自动驾驶"模式。

```python
# 示例：每4小时生产3条视频
scheduler.add_interval_job(
    func=produce_video_batch,
    hours=4,
    kwargs={
        "count": 3,
        "topics": ["trending", "seasonal", "evergreen"],
        "platforms": ["douyin", "xiaohongshu"]
    }
)
```

**调度模式**：

| 模式 | 说明 | 应用场景 |
|------|------|----------|
| **固定间隔** | 每N小时 | 日常内容生产 |
| **Cron表达式** | 复杂调度模式 | 周期性活动 |
| **事件驱动** | 由热点触发 | 实时内容 |
| **周期性** | 每日/每周/每月 | 内容日历 |

### 📦 批量生产引擎

**问题**：逐个制作视频效率低，无法规模化。

**解决方案**：并行批量处理，智能任务分配。

```python
# 示例：批量生产50个视频
batch_tasks = [
    {"topic": "产品展示", "template": "ecommerce.yaml"},
    {"topic": "客户评价", "template": "testimonial.yaml"},
    {"topic": "教程指南", "template": "tutorial.yaml"},
    # ... 更多任务
]

results = await batch_producer.produce(
    tasks=batch_tasks,
    parallel_workers=4,
    quality_threshold=0.8
)
```

## 🚀 快速开始

### 安装

```bash
# 克隆仓库
git clone https://github.com/kaka2022/video-maker.git
cd video-maker

# 创建虚拟环境
python -m venv venv
source venv/bin/activate  # macOS/Linux

# 安装依赖
pip install -r requirements.txt

# 配置环境
cp config.toml.example config.toml
# 编辑 config.toml 填入你的 API Key
```

### 基础用法

```bash
# 单视频生成
python video_maker.py --topic "你的主题" --images 素材/

# 定时生产（每4小时）
python video_maker.py --schedule "0 */4 * * *" --count 3

# 批量生产
python video_maker.py --batch batch_tasks.json --parallel 4

# 启动Web界面
python web/server.py
```

### 高级配置

```toml
# config.toml

[scheduler]
enabled = true
interval_hours = 4
batch_size = 3
topics = ["trending", "seasonal", "evergreen"]

[asset]
local_pool = "assets/"
ai_supplement = true
deduplication = true
quality_threshold = 0.8

[distribution]
platforms = ["douyin", "xiaohongshu"]
auto_publish = true
schedule_publish = true
```
