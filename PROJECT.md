# 团购视频生成器 - 项目说明

## 项目简介

**团购视频生成器**是一个专为本地生活商家设计的自动化短视频制作工具。

### 解决的问题

- ❌ 手动制作短视频耗时耗力
- ❌ 不会剪辑软件
- ❌ 请人制作成本高
- ❌ 内容创作困难

### 解决方案

- ✅ 输入主题关键词，自动生成视频
- ✅ AI 智能文案，自然流畅
- ✅ 专业转场效果，媲美人工制作
- ✅ 批量生成，效率提升 10 倍

## 核心优势

### 1. 简单易用
```bash
# 一行命令生成视频
python video_maker.py --topic "你的主题"
```

### 2. 专业效果
- Ken Burns 特效（图片缩放平移）
- 智能转场（平滑过渡）
- 字幕动画（10+ 样式）
- 背景音乐（自动匹配）

### 3. 高效批量
```bash
# 批量生成 10 个视频
python video_maker.py --batch batch_tasks.json
```

### 4. 灵活定制
- 商家模板（YAML 配置）
- 字幕样式（10+ 种类）
- TTS 引擎（Edge/MiMo）
- 素材来源（本地/Pexels）

## 配置示例

### 商家模板 (templates/shop.yaml)

```yaml
merchant:
  name: "你的店铺名称"
  full_name: "你的店铺全称"
  address: "你的店铺地址"
  phone: "你的联系电话"
  contact: "联系人姓名"
  hours: "10:00-22:00"

regions:
  - "你的城市"
  - "你的区域"

services:
  服务1:
    - name: "基础服务"
      price: 100
      unit: "元/次"
    - name: "高级服务"
      price: 200
      unit: "元/次"

selling_points:
  - "你的卖点1"
  - "你的卖点2"
  - "你的卖点3"

hashtags:
  - "#你的行业标签"
  - "#你的地区标签"
  - "#你的品牌标签"
```

### 配置文件 (config.toml)

```toml
[api]
# AI 文案生成（必须）
xiaomi_api_key = "YOUR_API_KEY"
xiaomi_base_url = "https://token-plan-cn.xiaomimimo.com/v1"

[pexels]
# 素材视频下载（可选）
api_key = "YOUR_PEXELS_KEY"

[video]
# 视频参数
width = 1080
height = 1920
fps = 30
img_duration = 4
transition_duration = 0.5
subtitle_style = 8

[audio]
# 音频参数
tts_engine = "edge_tts"
edge_tts_voice = "zh-CN-XiaoxiaoNeural"
bgm_volume = 0.12

[batch]
# 批量参数
max_workers = 4
output_dir = "输出"
```

## 常见问题

### Q: 需要什么系统？
A: 支持 macOS、Linux、Windows，需要 Python 3.8+ 和 FFmpeg。

### Q: 需要付费吗？
A: 
- Edge TTS：免费
- MiMo API：需要 API Key（小米提供）
- Pexels API：免费额度足够

### Q: 视频时长可以调整吗？
A: 可以，通过 `config.toml` 中的 `video.img_duration` 调整每张图片展示时长。

### Q: 可以用自己的图片吗？
A: 可以，使用 `--images` 参数指定图片目录。

### Q: 支持横版视频吗？
A: 可以，修改 `config.toml` 中的 `video.width` 和 `video.height`。

### Q: 批量生成有数量限制吗？
A: 没有硬性限制，建议根据机器性能设置合理的 `batch.max_workers`。
